Project Overview

ShopHub is a modern React-based e-commerce web application built with Vite, React Router, and Context API for state management. It provides a complete shopping experience with product browsing, search functionality, shopping cart, and wishlist features.

Key Features

  • Product Catalog - Browse products with category filtering
  • Product Search - Real-time search by name or description
  • Shopping Cart - Add/remove items with localStorage persistence
  • Wishlist - Save favorite products for later purchase
  • Toast Notifications - User feedback for all actions
  • Newsletter Subscription - Email subscription form
  • Contact Form - Customer support contact form

Tech Stack

State Management

CartContext

Property/Method Description
items Array of cart items
addToCart(product, quantity) Add item to cart
removeFromCart(productId) Remove item from cart
updateQuantity(productId, quantity) Update item quantity
clearCart() Empty the cart
totalItems Total item count
totalPrice Total price

WishlistContext (NEW)

Property/Method Description
items Array of wishlist items
addToWishlist(product) Add to wishlist
removeFromWishlist(productId) Remove from wishlist
isInWishlist(productId) Check if product is saved
totalItems Total saved items

ToastContext (NEW)

Property/Method Description
addToast(message, type) Show notification (success/error/info)

Responsive Design

The application is fully responsive with a breakpoint at 768px:

  • Mobile-friendly navigation (hidden on mobile)
  • Stacked product grids
  • Touch-friendly buttons (minimum 44px touch targets)
  • Adaptive layouts for all pages
  • Horizontal scrolling category filters on mobile

Data Persistence

The following data is persisted in localStorage:

  • cart - Shopping cart items (JSON)
  • wishlist - Saved wishlist items (JSON)

New Features Summary

1. Wishlist System

Users can now save products to a wishlist for later purchase. The wishlist is accessible from the header icon and persists across sessions.

2. Product Search

Real-time search functionality filters products by name or description. Works in combination with category filters.

3. Toast Notifications

User feedback system that shows notifications for:

  • Adding products to cart
  • Adding/removing from wishlist
  • Newsletter subscription
  • Contact form submission

4. Newsletter Subscription

Email subscription form on the homepage for marketing communications.

5. Contact Form

Customer support contact form with name, email, and message fields.
